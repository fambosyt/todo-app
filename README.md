# 📝 To-Do App

Eine einfache und minimalistische To-Do Liste im Browser – komplett mit HTML, CSS und JavaScript.

## 🚀 Features

- Aufgaben hinzufügen
- Aufgaben abhaken oder löschen
- Speicherung im LocalStorage
- Keine Installation nötig
